# ChessEver

**Following chess should be easier.**

ChessEver helps chess fans and players follow live tournaments, track favorite players, watch games, and work with chess databases across mobile and desktop.

- [Visit ChessEver](https://chessever.com)
- [Explore ChessEver Desktop](https://chessever.com/desktop)

## Open-source projects

- [ChessEver mobile](https://github.com/Chessever/chessever-frontend) — the Flutter client for following live chess tournaments, players, and games.
- [ChessEver Desktop](https://github.com/Chessever/chessever_frontend_desktop) — the desktop client for live chess, PGN workflows, databases, and preparation.
- [ChessEver TUI](https://github.com/Chessever/chessever-tui) — an experimental terminal chess interface.

ChessEver is built by chess players to help everyone **follow chess better**.
